How to get the file to import

  1. In ShipStation, search New Orders
  2. Pick any recent order
  3. Copy one of its SKUs
  4. Paste it into IMU
  5. Click the blue underlined Quantity Committed link
  6. Select ECOMM Viewer
  7. Set the date range and store number
  8. Click Export Kpad at the top. Opens in Excel. Save as an .xls to your computer
  9. Drag that file into the import box

How to get the open orders file

  1. In ShipStation, go to Orders
  2. Filter to whatever counts as "still open" for your workflow — typically Awaiting Shipment plus On Hold (backorders usually sit in On Hold)
  3. Select all the orders in that view
  4. Use the Export action to save a .csv
  5. Drag that file into the box above

Only an order number column is required — extra columns are ignored. If your export uses different column headers than expected, let me know and I'll adjust the matching.
