Fill missing UPCs and MFG codes in a Salesforce export using a Compass master file. Preview and download the enriched CSV. Nothing leaves your browser.

Drop CSV or Excel here or browse
Auto-detects columns named SKU, UPC, MFG, MPN, etc. Blank, 0, or N/A cells count as missing.

2 Fill from Compass

Drop Compass CSV or Excel or browse
Looks up each SKU in the Compass export. Fills missing UPC and MFG code values, and corrects any existing value that doesn't match Compass. Rows still missing data are highlighted; corrected rows are marked separately.
⚠ Don't open CSVs in Excel. It silently strips leading zeros and mangles scientific-notation SKUs. Use Google Sheets or import as Text.
