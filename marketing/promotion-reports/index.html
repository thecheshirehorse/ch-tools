Import an export

Paste the export from the Promotions dashboard (tab or comma separated), or upload the file. Revenue Per Order, Units Per Order, and Conversion Rate are always recalculated from the raw counts here — the export's own ratio columns are ignored, so an averaging error can't sneak back in.

Backup & restore

This tool has no server — everything lives in this browser's local storage. Export a backup after each import and commit it to your GitHub repo so the history survives a cleared cache or a new device.

Export

Download the full dataset as JSON. Commit this file to your repo (e.g. data.json) so it's versioned alongside the tool.

Restore

Load a previously exported JSON backup. Rows are merged in — matching Promotion ID + Site will be updated, not duplicated.

Reset
